/*
 *  To Do List App
 *  By: André Rocha
 *
 * How to setup:
 * Install latest php, apache and mysql. (On windows you can install wamp stack, on linux lamp stack).
 * Extract todolist folder on your www web server folder. 
 * Run your server and mysql.
 * Go to your www web folder, run 'composer dump-autoload', then 'php artisan migrate' to create database tables.
 * To reset BD, just run 'php artisan migrate:rollback', then 'php artisan migrate'.
 * Open "localhost/todolist/public" url in your browser to open application.
*/
